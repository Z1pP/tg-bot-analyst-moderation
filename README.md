По ТЗ

Вводные:
- чат на 2.5К человек
- модераторы (N никнеймов)
- админы (N никнеймов)

Необходимо:
- чекать количество соообщений за день на заданных аккаунтах
- чекать среднее количество сообщений за N времени (час/3 часа/6 часов/12 часов/заданное время) на заданных аккаунтах
- чекать разницу во времени ответа, если происходит реплай (ответ на сообщение) на заданных аккаунтах
- чекать период отсутствия (разница между сообщением условно в 13.05 и 17.14)

Объясню для чего нужен бот. В чате около 2500 человек (количество может расти в зависимости от чата) модераторы отвечают на вопросы пользователей, помогают им будет ответом на сообщения. Пользователя могут тэгать модераторов и задавать им вопросы. Модераторы отвечают на вопросы, фильтруют чат и следят за порядком. 

Бот нужен для сбора статистика о том как работает модерация


- проверка активности модерации входит только в период рабочего дня
